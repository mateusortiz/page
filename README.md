Page WebDev Group
====

### Project

Necessário ter Node.js Instalado

- Clone o projeto `git clone git@github.com:webdevbr/page.git`
- Entrar no diretório `cd page`
- Instalar as dependências `[sudo] npm install´
- Rodar Grunt `grunt build´
